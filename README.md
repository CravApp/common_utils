# common_utils.dart

Dart common utilities with no io or browser code for shared tekartik projects


## Usage

In your `pubspec.yaml`:

```yaml
dependencies:
  tekartik_common_utils:
    git:
      url: git://github.com/tekartik/common_utils.dart
      ref: dart2
    version: '>=0.9.8'
```
